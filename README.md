# 🧠 The Hidden Layer: Neural Network Architect

An interactive Machine Learning simulation designed to teach the inner workings of a **Neuron**. You play as a Digital Architect configuring a single-layer neural network to recognize handwritten digits. By adjusting weights and biases, you'll learn how raw signals are transformed into intelligent decisions using activation functions[span_0](start_span)[span_0](end_span)[span_1](start_span)[span_1](end_span).

## 🎓 Learning Objectives

This project focuses on teaching:
* **Neuron Anatomy:** Understanding the relationship between Inputs, Weights, and Biases[span_2](start_span)[span_2](end_span)[span_3](start_span)[span_3](end_span).
* **Weighted Sums:** The fundamental calculation behind every neural connection: $(Input \times Weight) + Bias$[span_4](start_span)[span_4](end_span).
* **Activation Functions (ReLU):** Learning how non-linear functions like ReLU ($max(0, x)$) allow networks to solve complex problems[span_5](start_span)[span_5](end_span).
* **Feature Extraction:** How pixel intensity is converted into a classification (e.g., Identifying a '0' vs a '1')[span_6](start_span)[span_6](end_span).
* **The Hidden Layer:** Visualizing how neurons act as "feature detectors" within a larger architecture[span_7](start_span)[span_7](end_span).

---

## ✨ Features

* **Digit Recognition Scenario:** Simplifies computer vision concepts into a single, tunable neuron interaction.
* **ReLU Implementation:** Includes a functional Rectified Linear Unit activation to demonstrate signal thresholds.
* **Real-time Processing:** Provides a step-by-step breakdown of the weighted sum before and after activation.
* **Architect Feedback:** Offers instant evaluation of your parameters to help you understand why a signal was lost or detected.
